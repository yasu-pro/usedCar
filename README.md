### 中古外車サイト

#サイトURL：https://usedcar.yasuthework.xyz/
#コード：https://github.com/yasu-pro/usedCar
#制作日数：約1カ月
#使用技術：HTML、CSS、JavaScript、PHP、MySQL、Git、GitHub
#ターゲット：20 代後半~30 代前半の男性 一般消費者
#サービス内容：検索メニューで調べたいメニュー(メーカー、車種、走行距離、年数)を選んでクリックするとそれに一致した車が表示されます。
さらに、表示された車をクリックすることで画像を拡大できます。
#デザインの趣旨：ターゲットが情報端末に慣れていることを想定し、作成。デザインについては、外車がメインになるので黒をベースに落ち着きがあり高級感のある雰囲気を出しました。
#改善点：画像が重く、動作が遅くなっていること。(画像のサイズの変更をする)
